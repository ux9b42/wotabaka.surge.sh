Astrological signs can be represented using Unicode symbols. Here are the Unicode representations for each zodiac sign:

  • Aries: ♈ (&#9800;)
  • Taurus: ♉ (&#9801;)
  • Gemini: ♊ (&#9802;)
  • Cancer: ♋ (&#9803;)
  • Leo: ♌ (&#9804;)
  • Virgo: ♍ (&#9805;)
  • Libra: ♎ (&#9806;)
  • Scorpio: ♏ (&#9807;)
  • Sagittarius: ♐ (&#9808;)
  • Capricorn: ♑ (&#9809;)
  • Aquarius: ♒ (&#9810;)
  • Pisces: ♓ (&#9811;)

These can be used in HTML or other systems that support Unicode encoding.
